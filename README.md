# BUWASA GIS–EPANET Hydraulic Model Visualization

## Project Description

This repository contains the BUWASA Water Distribution Network hydraulic model developed using Geographic Information Systems (GIS) and EPANET hydraulic modelling. The project demonstrates how GIS network data can be integrated with EPANET-JS to simulate and visualize hydraulic conditions within the BUWASA water distribution network.

The hydraulic model provides interactive visualization of pressure, flow, velocity, and head loss, allowing users to better understand the hydraulic behaviour of the network and assess areas that may have potential water loss risk.

This project was developed as part of a Bachelor of Science research project at Ardhi University.

---

## Project Title

**Assessment of Potential Water Loss Risk Areas in the BUWASA Water Distribution Network Using GIS and EPANET-JS Hydraulic Modelling**

---

## Project Objectives

The project aims to:

- Integrate BUWASA network data into an EPANET hydraulic model.
- Simulate hydraulic conditions of the BUWASA water distribution network.
- Visualize hydraulic simulation results using EPANET-JS.
- Support the assessment of potential water loss risk areas based on hydraulic conditions.

---

## Project Features

- GIS-based water distribution network.
- EPANET hydraulic simulation.
- Interactive visualization using EPANET-JS.
- Pressure distribution visualization.
- Flow distribution visualization.
- Velocity distribution visualization.
- Head loss distribution visualization.
- Interactive inspection of network components.
- Support for hydraulic network analysis.

## Download the Hydraulic Model

Download the BUWASA hydraulic model from:

https://raw.githubusercontent.com/QGIS-isack/EPANET-JS/main/buwasa.zip

After downloading:

1. Extract the ZIP file.
2. Locate **buwasa.inp**.
3. Open it using EPANET-JS or EPANET Desktop.

---

## Software Requirements

The model can be used with:

- EPANET-JS
- EPANET 2.2 Desktop

---

## How to Use

### Using EPANET-JS

1. Download **buwasa.zip** from this repository.
2. Extract the file to obtain **buwasa.inp**.
3. Open **https://app.epanetjs.com**.
4. Sign in to your EPANET-JS account.
5. Upload **buwasa.inp**.
6. Run the hydraulic simulation.
7. Explore the network interactively.

The following hydraulic parameters can be visualized:

- Pressure
- Flow
- Velocity
- Head Loss

---

## Model Outputs

The hydraulic model produces:

- Pressure (m)
- Flow (L/s)
- Velocity (m/s)
- Head Loss (m/km)

These outputs assist in understanding the hydraulic performance of the BUWASA water distribution network and identifying sections that may require further field investigation for potential water loss.

---

## Important Note

EPANET-JS requires users to have an account and be logged in before opening or running hydraulic models.

Because of this authentication requirement, the model cannot be accessed directly through a shared EPANET-JS URL by users who are not logged in.

To use the model:

- Download **buwasa.zip** from this repository.
- Extract **buwasa.inp**.
- Log in to EPANET-JS.
- Upload the model manually.

---

## Limitations

- The hydraulic model identifies potential water loss risk areas based on simulated hydraulic conditions.
- It does not directly detect or confirm physical leaks.
- Field measurements and site inspections are required to verify suspected leakage locations.
